# NoteTracker

This noteTracker will allow users to easily add and delete notes, all while being able to see an ongoing record of ones previously written.

## Features

- Heroku deployment
- express.js
- back-end db routes

## Installation

Install dependencies with npm

```cmd-line
  npm install
```

Requires express.js
Requires uuid
